# 穷观阵 - 多游戏抽卡记录分析器

> 乾坤清策，否极泰来

一款支持多款游戏的抽卡记录分析工具，提供保底计算、统计分析、图表展示等功能。

## 支持的游戏

| 游戏 | 自动获取 | 手动导入 | 卡池类型 |
|------|---------|---------|---------|
| 原神 | ✅ | ✅ UIGF/小黑盒 | 角色池、武器池、集录祈愿、常驻池、新手池 |
| 崩坏：星穹铁道 | ✅ | ✅ UIGF/小黑盒 | 角色跃迁、光锥跃迁、联动跃迁、常驻跃迁、新手跃迁 |
| 绝区零 | ✅ | ✅ UIGF/小黑盒 | 独家频段、音擎频段、独家重映、音擎回响、邦布频段、常驻频段 |
| 鸣潮 | ✅ | ✅ 小黑盒 | 角色唤取、武器唤取、常驻唤取、新旅唤取、新手唤取 |
| 明日方舟 | ✅ | ✅ 小黑盒 | 标准寻访、中坚寻访、独立寻访 |
| 终末地 | ✅ | ✅ | 特许寻访、辉光庆典、基础寻访、武库申领、启程寻访 |

## 功能特性

### 📊 数据获取
- **自动获取**：从游戏缓存中自动提取抽卡记录URL
- **登录获取**：支持鹰角账号登录（明日方舟/终末地）
- **手动导入**：支持 JSON（UIGF/小黑盒格式）、Excel、CSV 文件导入

### 📈 统计分析
- **保底计算**：实时计算当前保底进度、距离硬保底抽数、当前出货概率
- **出货统计**：5星/6星数量、平均出金抽数、最欧/最非记录
- **50/50统计**：UP胜率、歪的次数（明日方舟除外）
- **概率预测**：逐抽出货概率曲线、滑块预测

### 📉 图表展示
- 出货抽数分布直方图
- 保底概率曲线
- 各星级数量统计
- 卡池投入占比饼图
- 月度抽卡趋势
- 50/50统计饼图

### 🔧 其他功能
- **多账号支持**：每个游戏可管理多个账号
- **增量同步**：只获取新记录，不重复拉取
- **数据备份**：自动备份数据库
- **手动添加**：支持手动输入抽卡记录
- **卡池管理**：自定义卡池显示/隐藏、排序

## 安装与运行

### 环境要求
- Python 3.8+
- Windows 10/11

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行程序
```bash
python main.py
```

### 打包成 exe
```bash
python build.py
```

## 使用说明

### 自动获取（推荐）
1. 打开游戏，进入抽卡记录页面
2. 等待记录加载完成
3. 切回本程序，点击「开始获取」

### 登录获取（明日方舟/终末地）
1. 点击「登录获取」按钮
2. 输入鹰角账号密码
3. 登录成功后自动获取记录

### 手动导入
1. 点击「导入 JSON」
2. 选择导出的 JSON 文件
3. 支持 UIGF 标准格式和小黑盒导出格式

## 常见问题

### Q: 为什么自动获取失败？
A: 请确保：
1. 游戏已打开
2. 已进入抽卡记录页面
3. 记录已加载完成
4. 切回程序后重新点击获取

### Q: authkey 过期了怎么办？
A: 重新打开游戏，进入抽卡记录页面，然后切回程序重试。

### Q: 支持国际服吗？
A: 支持，程序会自动检测服务器区域。

### Q: 数据会上传到服务器吗？
A: 不会，所有数据都保存在本地数据库中。
